# Integrations

Generated by `tools/gen_integrations.py` from `manifest.json` — do not edit by hand.

The library ships **100 prompts** and **84 skills** across **7 categories** (31 skills carry a validation/calculation script). Everything is plain markdown plus stdlib Python, so it runs under any LLM agent — no provider lock-in.

Machine-readable index: [`manifest.json`](../manifest.json). Task coverage: [`prompt-skill-map.md`](prompt-skill-map.md).

## Any agent

Point your agent at `skills/<category>/<name>/SKILL.md`. It is plain markdown with a small YAML header — no runtime-specific syntax.

## Terminal / scripts

Use the bundled CLI (stdlib Python, zero deps):

```
python tools/skills_cli.py list --with-scripts
python tools/skills_cli.py search latency
python tools/skills_cli.py cat promotion-case | your-agent
```

## Claude Code

Add to your `CLAUDE.md`:

```
skillsDir: skills/
```

## GitHub Copilot

Copy the relevant skill folder into `.github/copilot-instructions/`.

## Cursor

Reference a skill inline via `@skills/<name>/SKILL.md`, or add the path to `.cursorrules`.
