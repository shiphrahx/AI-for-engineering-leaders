# Weekly Comms

The communication layer of engineering leadership. These prompts cover the recurring messages you write every week — plus the ad-hoc announcements that need to land well.

## When to Use These

- **Monday morning:** Grab the exec status update prompt to synthesise your team leads' updates
- **After a launch:** Use the internal or external announcement prompt
- **After an incident:** Use the stakeholder comms prompt while the timeline is fresh
- **After any meeting:** Turn messy notes into clear recaps with action items
