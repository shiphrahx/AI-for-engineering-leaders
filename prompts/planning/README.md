# Planning Artifacts

The documents that shape what gets built and when. These prompts help you produce the planning artifacts that engineering orgs run on — roadmaps, RFCs, retros, decision records, and operational playbooks.

## When to Use These

- **Start of quarter:** Roadmap and OKR drafting
- **Before a big project:** Pre-mortem, capacity planning, RFC outline
- **After a sprint:** Retro summary, sprint planning breakdown
- **After an incident:** Postmortem
- **At a decision point:** Build vs. buy, architecture decision record
