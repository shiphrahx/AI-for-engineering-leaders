# Hiring Pipelines

The end-to-end process of finding, evaluating, and closing engineering candidates. These prompts cover everything from pipeline strategy through to offer negotiation — not just the interview itself, but the system around it.

## When to Use These

- **Opening a role:** Hiring plan, sourcing strategy, recruiter kickoff
- **Running a pipeline:** Candidate evaluation summary, interview debrief facilitation
- **Closing candidates:** Offer justification, candidate pitch, rejection email
- **Improving the process:** Pipeline analytics, interviewer calibration, candidate experience audit
