# Contributing

Contributions are welcome. If you have a prompt that's worked well for you, add it.

## How to contribute

1. Fork the repo
2. Add or edit prompts — follow the existing format (Situation, The Prompt, Example Input, Example Output, Tuning Notes)
3. Open a pull request with a short description of what you added and why it's useful

There's no formal review process. If the prompt is specific, practical, and follows the format, it's in.
